[Conexion Base de Datos]

1. [Paquetes a utilizar]

   -> pip install python-dotenv pymysql

2. [Archivo .env]

  -> Crear archivo .env con la siguiente infomación
        * MYSQL_SERVER=172.17.183.129
        * MYSQL_DATABASE=conexionFastAPI
        * MYSQL_USER=root
        * MYSQL_PASSWORD=pass123      
        * MYSQL_PORT=3307

3. [Carpeta core]

  -> Dentro de la carpeta [core] crear los siguientes archivos.
     * __init__.py: se utiliza para indicar que un directorio debe ser tratado como un paquete, para que el código dentro de ese directorio se pueda importar como un módulo
     * config.py: Configuración principal para la conexion a la BD 

4. [Carpeta db]
 
  -> Dentro de la carpeta [db] crear los siguientes archivos:
    * __init__.py: se utiliza para indicar que un directorio debe ser tratado como un paquete, para que el código dentro de ese directorio se pueda importar como un módulo
    * session.py: gestionar sesiones de base de datos en una aplicación FastAPI que utiliza SQLAlchemy como la herramienta ORM (Mapeo Objeto-Relacional)
    * base.py: Esta clase es la base para todas las clases de modelo.

5.- Modificar Archivo main.py para agregar la Conexión y Creación   

        # Clase principal que se utiliza para crear una aplicación FastAPI
        from fastapi import FastAPI
        #  Clase base declarativa para los modelos de SQLAlchemy
        from app.db.base import Base
        # Este es el motor de SQLAlchemy que se conecta a la base de datos
        from app.db.session import engine

        # Inicializa una nueva aplicación en FastAPI.
        app = FastAPI()

        # Crea todas las tablas definidas en los modelos que heredan de Base en la base de datos. 
        Base.metadata.create_all(bind=engine)

        app.include_router(api_router)
